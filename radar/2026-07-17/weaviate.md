**Update 07/2026:** Notable developments over the past year:

- **v1.37** (April 2026) ships a **built-in MCP server** - agents can query Weaviate directly as an [MCP](/architecture-pattern/mcp_model_context_protocol/) tool without custom glue code - plus diversity search (MMR) and query profiling.
- **v1.36** (March 2026): server-side batching GA and a new vector index (HFresh, preview).
- "Agent Skills" (February 2026) target coding agents working with Weaviate.

The direction is clear: Weaviate positions itself as retrieval backend for agentic systems, not only as a vector store. Ring stays adopt.
