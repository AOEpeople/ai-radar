**Update 07/2026:** Gitleaks is officially **feature-complete** - the maintainer merges no new features anymore, future releases are security patches only (last release v8.30.1, March 2026). The original author has started a successor project, [Betterleaks](/data-features/betterleaks/), which we track separately.

Gitleaks remains stable and our recommendation for secret scanning stands unchanged. No migration needed - but evaluate the successor before building new, extensive custom-rule setups.
