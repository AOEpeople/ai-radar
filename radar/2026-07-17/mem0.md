**Update 07/2026 (assess → trial):** Mem0 has established itself as the most visible open-source memory layer for agents and is worth trialling.

The framework is now in v2 and includes a more token-efficient memory algorithm (April 2026) and native integrations into agent frameworks such as CrewAI, Flowise and Langflow.
