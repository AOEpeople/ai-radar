OpenCode (SST) is the leading open-source coding agent (MIT license) - and the provider-neutral counterpart to [Claude Code](/tools/claude_code/) and [OpenAI Codex](/tools/openai_codex/).

## Key Features

- **Provider-agnostic**: works with 75+ model providers including local models - no vendor subscription required, models are billed separately (or run on your own hardware).
- Terminal UI as the core, plus desktop app and IDE extension; LSP integration for language-aware edits.
- **Plan/build modes**: plan mode is read-only for reviewing the agent's approach before it touches files.
- Privacy stance: no storage of code or context by the tool vendor.
- Optional "Zen" offering: curated, benchmarked model access with pay-as-you-go pricing.

## Assessment

The strongest option when model neutrality, data sovereignty, privacy or cost control matter - including fully local setups with open-weight models.
